Hi, I'm Yash

I designed this blog-layout as a practice project.

This is a static website. Here I focused mainly on creating a responsive design, so as to enable adaptive rendering on various devices.

Technologies used

My aim was to practice CSS concepts such as Flex-box and media-queries. Hence, didn't focus much on Javascript. Didn't use React here.

The blog content is imported from a separate file where it is stored in JSON format. It allows me some flexibility to alter/update the content. However, I eventually plan to connect it to an API to make it more dynamic, and also allow users to create new posts.
